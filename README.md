# Numerical Analysis Final Project: Bloch Theorem Visualization

## Project Description
This project utilizes the Monte Carlo Method to simulate and visualize the Bloch Theorem and energy band structures in solid-state physics.
The program calculates the energy eigenvalues of electrons under a periodic potential and generates a dynamic GIF illustrating the convergence process.

## Requirements
This project is using Python 3，the necessary package：
* `numpy`
* `matplotlib`
* `scipy`
